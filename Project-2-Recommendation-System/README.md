# AI Recommendation System



### Project Overview



This project is a simple AI Recommendation System developed using Python. It recommends items to users based on their selected interests or preferences. The system demonstrates the basic concept of recommendation logic by matching user input with predefined categories.



### Features



\- Accepts user input for interests or preferences.

\- Matches user preferences using simple logic.

\- Displays relevant recommendations.

\- User-friendly command-line interface.



### Technologies Used



\- Python 3



### Project Structure



AI\_Recommendation\_System/

│── recommendation.py

│── README.md



### How to Run the Project



1\. Install Python 3 on your system.

2\. Download or clone this repository.

3\. Open the project folder in VS Code or any Python IDE.

4\. Run the following command:



python recommendation.py



5\. Enter your preferred category when prompted to receive recommendations.
