# Rule-Based AI Chatbot

# 

### Project Overview



This project is a simple Rule-Based AI Chatbot developed using Python as part of the Artificial Intelligence Internship at DecodeLabs. The chatbot interacts with users by responding to predefined inputs using conditional statements ("if-elif-else").



### Objective



The objective of this project is to understand the fundamentals of Artificial Intelligence by implementing a basic chatbot that simulates human conversation through rule-based logic.

### 

### Features



\- Responds to greetings such as "Hello" and "Hi"

\- Answers predefined questions

\- Handles unknown inputs with a default response

\- Runs continuously until the user enters an exit command

\- Uses simple "if-elif-else" decision-making logic



### Technologies Used



\- Python 3

\- Visual Studio Code (or any Python IDE)

##### How to Run the Project



1\. Install Python 3 on your system.

2\. Download or clone this repository.

3\. Open the project folder in your preferred Python IDE.

4\. Run the following command:



python chatbot.py



5\. Start chatting with the bot.

6\. Type "bye" or "exit" to end the conversation.
